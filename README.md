# ☕ Coffee Shop Sales Analysis – Power BI Dashboard

## 📌 Project Overview
This project presents an **interactive Power BI dashboard** built to analyze coffee shop sales performance.  
It provides insights into **revenue trends, product performance, customer behavior, and time-based analysis** to support business decision-making.

---

## 🎯 Objectives
- Analyze total sales and profit performance
- Identify best-selling and low-performing products
- Understand customer purchase patterns
- Track daily, monthly, and yearly sales trends
- Support data-driven decisions for coffee shop management

---

## 📊 Dashboard Features
- 📈 **Sales Trends** (Daily / Monthly / Yearly)
- ☕ **Top Products by Revenue & Quantity**
- 💰 **Total Sales, Profit, and Average Order Value**
- 🕒 **Time-based Analysis (Hour, Day, Month)**
- 📍 **Category-wise & Product-wise breakdown**
- 🎯 Interactive slicers for easy filtering

---

## 🛠 Tools & Technologies
- **Power BI Desktop**
- **DAX (Data Analysis Expressions)**
- **Power Query (ETL)**
- **Data Modeling (Star Schema)**
